HTML5 Tags Learning

# Name of Tag Meaning
1 header Represents the header of a document or a section.
2 label Defines a label for an input control.
3 html Defines the root of an HTML document.
4 nav Defines a section of navigation links.
5 img Represents an image.
6 p Defines a paragraph.
7 footer Represents the footer of a document or a section.
8 head Defines the head portion of the document that contains information about the document such as title.
9 h1 - h6 Defines HTML headings.
10 input Defines an input control.
11 section Defines a section in a document.
12 table Defines a table.
13 title Defines a title for the document.
14 style Defines style information for a document.
